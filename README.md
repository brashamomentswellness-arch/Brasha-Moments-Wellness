# Brasha Moments Wellness Website

Professional website for Brasha Moments Wellness - A psychiatric, psychological, drug, and substance abuse counseling treatment center in Kenya.

## 🚀 Quick Deploy to Vercel

### Method 1: Web Interface (Easiest - No Coding)
1. Go to [vercel.com](https://vercel.com) and sign up (free account)
2. Click "Add New" → "Project"
3. Drag and drop all files from this folder
4. Click "Deploy"
5. Your site will be live at: `https://brashamoments.vercel.app` (or similar)

### Method 2: Vercel CLI
```bash
npm i -g vercel
cd /path/to/deployment-package
vercel
```

## 🌐 Custom Domain Options

### Free Vercel Domains:
- `brashamoments.vercel.app`
- `brasha-moments-wellness.vercel.app`
- `brashamomentswellness.vercel.app`

### Add Custom Domain Later:
You can add domains like:
- `brashamoments.co.ke`
- `brashamoments.com`
- `brashamomentswellness.org`

**Steps to add custom domain:**
1. Go to your Vercel dashboard
2. Select your project
3. Go to Settings → Domains
4. Add your custom domain
5. Update your domain's DNS settings as instructed
